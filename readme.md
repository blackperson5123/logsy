TaskManager

Консольное приложение для управления задачами с демонстрацией структурированного логирования (Serilog) и трассировки.

Возможности
- Добавление задачи – название, описание, приоритет (Low/Medium/High).
- Удаление задачи – поиск по названию.
- Просмотр списка – с идентификаторами и подробной информацией.
- Логирование – одновременная запись в консоль и в файл с ротацией по дням.
- Трассировка – фиксация начала и окончания операций с замером времени выполнения.

Требования
- .NET 8.0
- NuGet‑пакеты:
  - Serilog
  - Serilog.Sinks.Console
  - Serilog.Sinks.File
  - Serilog.Formatting.Compact
